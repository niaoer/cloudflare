WordPress CloudFlare 插件汉化版
==========

### 说明

由于该插件并未提供国际化支持，所以选择了直接将 PHP 文件汉化。

### 安装方法

下载 `cloudflare.php` 和 `ip_in_range.php` 文件或者压缩包解压后，上传至 `wp-content\plugins\cloudflare` 文件夹中，启用插件。

需要注意的是，Cloudflare 配置界面位于插件菜单下，而不是常见的设置。
